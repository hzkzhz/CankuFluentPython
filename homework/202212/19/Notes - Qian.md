# Data Model

- Python 最好的品质之一是一致性。
- 数据模型其实是对 Python 框架的描述，它规范了这门语言自身构建模块的接口，这些模块包括但不限于序列、迭代器、函数、类和上下文管理器。
- collections.namedtuple可以构建类
- random.choice 随机选出一个元素的函数
