##### Using iter with a Callable
> iter(...)
    iter(iterable) -> iterator
    iter(callable, sentinel) -> iterator
```python
def d6():
    return randint(1, 6)
d6_iter = iter(d6, 1)
for roll in d6_iter:
    print(roll)
```
